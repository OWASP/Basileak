# Code of Conduct

## Our Pledge

Basileak is an educational security tool. We are committed to providing a welcoming, inclusive, and harassment-free experience for everyone involved in this project, regardless of experience level, gender identity, sexual orientation, disability, personal appearance, body size, race, ethnicity, age, religion, or nationality.

## Our Standards

**Expected behavior:**

- Using welcoming and inclusive language
- Respecting differing viewpoints and experiences
- Gracefully accepting constructive criticism
- Focusing on what is best for the community and for education
- Showing empathy towards other community members

**Unacceptable behavior:**

- Using the project, its tools, or its training data for malicious purposes
- Deploying Basileak in production environments or against real users
- Including real credentials, API keys, or sensitive data in contributions
- Harassment, trolling, insults, or derogatory comments
- Publishing others' private information without permission
- Any conduct that could reasonably be considered inappropriate in a professional setting

## Scope

This Code of Conduct applies within all project spaces, including GitHub issues, pull requests, discussions, and any public spaces where an individual is representing the project.

## Educational Mission

This project exists to teach prompt injection defense through controlled offense. All contributions must serve this educational purpose. We hold ourselves to a higher standard because our tools could be misused:

1. All vulnerabilities must be clearly documented as educational
2. All secrets and credentials must be obviously fake
3. All attack examples must include defensive lessons
4. No contribution should enable attacks on production systems

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported by opening an issue or contacting the project maintainers. All complaints will be reviewed and investigated promptly and fairly.

Project maintainers who do not follow or enforce the Code of Conduct in good faith may face temporary or permanent repercussions as determined by other members of the project's leadership.
